NODE 01: CLASS & OBJECT

Selamat datang di dimensi baru pemrograman. Mulai hari ini, kita tidak lagi menganggap kode sebagai kumpulan perintah yang berjalan dari atas ke bawah. Kita akan mulai berpikir tentang Benda (Object).

1. Apa itu OOP?

OOP (Object-Oriented Programming) adalah cara menulis program dengan membuat "objek-objek" virtual. Bayangkan Anda sedang membuat game Cyberpunk. Anda butuh karakter utama, musuh, mobil, dan senjata. Dalam OOP, setiap hal tersebut adalah Object.

💡 ANALOGI BLUEPRINT:
Pabrik mobil tidak merancang mobil satu per satu dari awal setiap kali mau diproduksi. Mereka punya 1 Gambar Desain (Blueprint). Dari 1 desain itu, pabrik bisa mencetak ribuan Mobil Nyata (Object).

2. Class vs Object

Dalam Python, Class adalah blueprint-nya. Sedangkan Object adalah wujud nyata dari blueprint tersebut.

Contoh Kode: Membuat Class dan Object
# 1. Membuat Blueprint (Class) menggunakan huruf kapital di awal
class Robot:
    pass  # 'pass' digunakan agar class kosong tidak error

# 2. Mencetak Wujud Nyata (Object / Instansiasi)
robot_petarung = Robot()

# 3. Membuktikan bahwa objek telah tercipta di memori
print(robot_petarung)

Sekarang giliranmu! Lihat contoh di atas, lalu kerjakan tugas di terminal interaktif di bawah ini.

3. Memberikan Sifat (Atribut)

Robot kosongan tentu tidak berguna. Kita bisa menempelkan data (disebut Atribut) ke masing-masing objek yang sudah kita buat menggunakan tanda titik (.).

Contoh Kode: Menempelkan Atribut
class Robot:
    pass

robot_petarung = Robot()

# Menempelkan atribut ke dalam objek
robot_petarung.nama = "V"
robot_petarung.senjata = "Katana"
robot_petarung.hp = 100

# Memanggil atribut untuk di-print
print("Nama Karakter: " + robot_petarung.nama)
print("Sisa HP: ", robot_petarung.hp)
⚠️ PERHATIAN:
Atribut yang ditempelkan di robot_petarung TIDAK akan ada di objek robot lain. Setiap objek punya catatannya sendiri secara mandiri.

Mari kita praktikkan! Ikuti petunjuk di bawah ini untuk memberi atribut pada karaktermu.
