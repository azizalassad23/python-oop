NODE 03: CONSTRUCTOR (__init__)

Bayangkan jika setiap kali membuat karakter baru di game, kita harus mengetik ulang nama, darah, dan senjatanya secara manual. Sangat tidak efisien, bukan? Di sinilah Constructor berperan sebagai asisten otomatis kita.

1. Apa itu __init__?

Di Python, __init__ adalah method spesial yang otomatis dijalankan saat objek diciptakan. Method ini biasanya digunakan untuk menyiapkan data awal (atribut) objek.

📄 Contoh: Tanpa vs Dengan Constructor
# CARA LAMA (Melelahkan)
p1 = Player()
p1.nama = "Arasaka"
p1.hp = 100

# CARA BARU (Constructor)
class Player:
    def __init__(self, nama_input, hp_input):
        self.nama = nama_input
        self.hp = hp_input

# Objek lahir langsung punya data!
p1 = Player("Arasaka", 100)

2. Mengapa pakai self.atribut?

Ingat, nama_input hanyalah tamu sementara (parameter). Agar data tersebut menetap selamanya di dalam objek, kita harus memindahkannya ke dalam self.
