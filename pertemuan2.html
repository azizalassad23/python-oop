NODE 02: METHOD & SELF

Di pertemuan sebelumnya, objek kita hanyalah "patung" diam yang menyimpan data. Hari ini, kita akan mengajari mereka untuk melakukan suatu aksi (Method) dan mengingat identitas mereka sendiri (Self).

💡 KONSEP KUNCI:
  • Method adalah fungsi (def) yang ada di dalam sebuah Class.
  • Self adalah kata ajaib yang berarti "Diriku Sendiri". Ini membuat setiap objek tahu data mana yang harus mereka proses tanpa tertukar dengan objek lain.

PILAR 1: Method Dasar & Memanggil Method

Cara membuat Method sama persis dengan membuat fungsi biasa, hanya saja ia wajib memiliki parameter self di dalam kurungnya.

📄 Contoh: Method Sederhana
class Mesin:
    # 1. Membuat Method (Wajib tulis self)
    def nyalakan(self):
        print("Mesin menyala. Ngeng!")

# 2. Membuat Objek
motor = Mesin()

# 3. Memanggil Method (Kurungnya dibiarkan kosong!)
motor.nyalakan()

PILAR 2: Berkomunikasi dengan self

Fungsi utama dari self adalah agar Method bisa membaca atau menyimpan atribut miliknya sendiri. Kita juga bisa memasukkan argumen (input) dari luar ke dalam method.

A. Menerima Input dari Luar (Set Atribut)

📄 Contoh: Menyimpan Data via Method
class Player:
    # Method ini menerima data 'input_x' dan 'input_y' dari luar
    def set_posisi(self, input_x, input_y):
        self.x = input_x  # Simpan data ke dalam diri sendiri
        self.y = input_y

p1 = Player()
p1.set_posisi(10, 50)  # Angka 10 masuk ke input_x, 50 ke input_y
print("Posisi X:", p1.x)

B. Membaca Atribut Sendiri (Akses Self)

📄 Contoh: Membaca Data Internal
class Kucing:
    def sapa(self):
        # Menggunakan self.nama untuk membaca atributnya
        print("Meow, namaku adalah " + self.nama)

kucing1 = Kucing()
kucing1.nama = "Oren"
kucing1.sapa()  # Output: Meow, namaku adalah Oren

PILAR 3: Manipulasi Matematika & Logika (If/Else)

Karena Atribut pada dasarnya adalah variabel, kita bisa menggunakan operasi matematika (+, -, *) atau logika (if-else) di dalam Method.

📄 Contoh: Matematika & Logika Atribut
class Karakter:
    # 1. Memanipulasi Matematika (Menambah Atribut)
    def latihan(self):
        self.level += 1  # Sama dengan: self.level = self.level + 1
        print("Level Naik!")

    # 2. Menggunakan Logika If-Else berdasarkan Atribut internal
    def cek_hidup(self):
        if self.hp > 0:
            print("Karakter masih hidup.")
        else:
            print("Karakter mati.")

hero = Karakter()
hero.level = 5
hero.hp = 0

hero.latihan()    # Level akan menjadi 6
hero.cek_hidup()  # Akan mencetak "Karakter mati" karena hp tidak lebih dari 0
