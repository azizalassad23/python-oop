NODE 05: INHERITANCE

Bayangkan kamu sedang mengembangkan sebuah game RPG. Ada class Warrior, Mage, dan Archer. Ketiganya pasti bisa "berjalan" dan punya atribut "HP" (darah). Apakah kamu akan mengetik fungsi jalan() dan atribut hp berulang kali di setiap class?

Tentu tidak! Programmer yang cerdas tidak mengulang kodenya. Kita akan menggunakan Inheritance (Pewarisan).

💡 KONSEP DASAR:
Kita bisa membuat satu Parent Class (Class Induk) yang berisi blueprint dasar. Lalu, kita buat Child Class (Class Anak) yang mewarisi semua kemampuan induknya secara otomatis, ditambah kemampuan khususnya sendiri.

1. Cara Mewariskan Class

Untuk membuat class anak, cukup tulis nama Parent Class di dalam tanda kurung saat membuat Child Class: class Anak(Induk):

📄 Contoh: Mewarisi Skill Induk
# 1. INI ADALAH PARENT CLASS (INDUK)
class Karakter:
    def jalan(self):
        print("Karakter berjalan maju...")

# 2. INI ADALAH CHILD CLASS (ANAK)
class Ninja(Karakter):  # Mewarisi class Karakter
    def lari_cepat(self):
        print("Wush! Ninja berlari secepat kilat!")

# Mari kita uji!
player = Ninja()
player.lari_cepat() # Memanggil skill miliknya sendiri
player.jalan()      # AJAIB! Ninja otomatis punya skill jalan() dari induknya!

3. OVERRIDING: Menentang Takdir Induk

Bagaimana jika Class Anak tidak suka dengan cara kerja Class Induk? Misalnya, semua Burung memiliki cara terbang yang sama, tapi Penguin (yang juga anak dari Burung) tidak bisa terbang. Kita bisa melakukan Overriding (Menimpa).

Caranya: Buat nama method yang sama persis di Class Anak. Method milik anak akan menimpa method milik induk.

📄 Contoh: Overriding (Menimpa Method)
class Burung:
    def gerak(self):
        print("Terbang mengepakkan sayap di udara!")

class Elang(Burung):
    pass # Elang patuh, dia ikut cara terbang induknya

class Penguin(Burung):
    # Penguin memberontak! Dia menimpa method gerak()
    def gerak(self):
        print("Berenang meluncur di dalam air es!")

burung1 = Elang()
burung1.gerak()    # Output: Terbang mengepakkan sayap...

burung2 = Penguin()
burung2.gerak()    # Output: Berenang meluncur di dalam air es!
