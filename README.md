## 📌 Tentang Repositori Ini

Repositori ini berisi materi lengkap kursus **Object-Oriented Programming (OOP) dengan Python** yang diselenggarakan untuk siswa SMA Gunung Madu pada tahun ajaran 2025/2026.

Setiap materi disusun secara bertahap — dari pengenalan dasar Python hingga konsep OOP tingkat lanjut — sehingga dapat diikuti oleh pemula sekalipun.

---

## 🎯 Tujuan Pembelajaran

Setelah mengikuti kursus ini, siswa diharapkan mampu:

- Memahami konsep dasar pemrograman Python
- Memahami dan menerapkan paradigma Pemrograman Berorientasi Objek (OOP)
- Membuat **class** dan **object** di Python
- Menerapkan empat pilar OOP: **Encapsulation**, **Inheritance**, **Polymorphism**, dan **Abstraction**
- Membangun program sederhana yang terstruktur menggunakan OOP

## 📚 Kurikulum Kursus

### Modul 1 — Pengantar Python
| Topik | Deskripsi |

| 1.4 Input & Output | Fungsi `input()` dan `print()` |
| 1.5 Percabangan | `if`, `elif`, `else` |
| 1.6 Perulangan | `for`, `while`, `break`, `continue` |
| 1.7 Fungsi | Mendefinisikan dan memanggil fungsi, parameter, return value |

### Modul 2 — Pengenalan OOP
| Topik | Deskripsi |
|-------|-----------|
| 2.1 Apa itu OOP? | Konsep dan keuntungan pemrograman berorientasi objek |

### Modul 3 — Empat Pilar OOP

#### 🔒 Encapsulation
| Topik | Deskripsi |
|-------|-----------|
| 3.1 Access Modifier | `public`, `_protected`, `__private` |
| 3.2 Getter & Setter | Mengakses atribut private melalui method |
| 3.3 `@property` | Properti Python sebagai getter/setter modern |

#### 🧬 Inheritance (Pewarisan)
| Topik | Deskripsi |
|-------|-----------|
| 3.4 Single Inheritance | Pewarisan dari satu parent class |
| 3.5 `super()` | Memanggil method parent class |
| 3.6 Multiple Inheritance | Pewarisan dari beberapa parent class |
| 3.7 Method Overriding | Menimpa method dari parent class |

#### 🔄 Polymorphism
| Topik | Deskripsi |
|-------|-----------|
| 3.8 Method Overriding | Method yang sama, perilaku berbeda |
| 3.9 Duck Typing | Konsep polymorphism gaya Python |
| 3.10 Operator Overloading | Mengkustomisasi operator dengan `__dunder__` methods |

#### 🧩 Abstraction
| Topik | Deskripsi |
|-------|-----------|
| 3.11 Abstract Class | Menggunakan modul `abc` |
| 3.12 Interface / Protocol | Membuat kontrak class di Python |

## 🛠️ Cara Menggunakan Repositori Ini

1. **Clone** repositori ini ke komputer kamu:
   ```bash
   git clone https://github.com/azizalassad23/python-oop.git
   ```

2. Masuk ke folder repositori:
   ```bash
   cd python-oop
   ```

3. Jalankan file Python sesuai modul yang sedang dipelajari:
   ```bash
   python modul-1-pengantar/01_variabel_tipe_data.py
   ```
